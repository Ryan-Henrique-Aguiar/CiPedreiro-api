# CiPedreiro API

API backend para gerenciamento de serviços de pedreiros / prestadores de serviços.  
Permite cadastro de clientes, provedores, serviços prestados, custos, pagamentos e controle de status.

## Tecnologias principais

- **Java 21** + **Spring Boot 3**
- **Spring Security** + **JWT** (autenticação e autorização)
- **Spring Data JPA** + **Hibernate**
- **PostgreSQL** (banco de produção)
- **Flyway** (migrações)
- **Lombok**, **MapStruct** (ou mappers manuais)
- **UUID** como ID das entidades
